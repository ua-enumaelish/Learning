FORMS

  • Компоненты форм довольно интересны своим специфическим поведением, а также особенностями взаимодействия с пользователем. Каждый front-end разработчик рано или поздно сталкивается с задачей - “добавить к проекту нестандартно выглядящий но вполне стандартно работающий компонент формы”. То есть часть формы которая собирает у пользователя данные но выглядит как нарисовано в дизайне.
    • Реализовать модель кастомного input
    • Реализовать модель кастомного checkbox
    • Реализовать модель кастомного radiobutton
    • Реализовать модель кастомного Select (сложно/не обязательно/полезно)
  • Каждую модель держать в отдельном файле. Создать проект (страничку index.html) к которому подключить все созданные модели. На главной странице должна быть форма состоящая из кастомных полей (минимум по 2 каждого). К форме добавить кнопку которая будет выводить (в консоль например) введенные в форму данные.
